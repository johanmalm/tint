# tint

## Why

Pursuing beauty.

## What

A panel inspired by tint2.

Try with:
- https://github.com/addy-dclxvi/tint2-theme-collections

## Dependencies

- meson, ninja, cmake
- qt6-base, qt6-wayland
- [layer-shell-qt]

[layer-shell-qt]: https://invent.kde.org/plasma/layer-shell-qt

## Build

```
meson setup build
meson compile -C build
```
